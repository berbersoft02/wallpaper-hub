# Structure des Wallpapers

Placez vos wallpapers dans cette structure :

```
public/wallpapers/
  ├── nom-personnage-1/
  │   ├── wallpaper1.jpg
  │   ├── wallpaper2.png
  │   └── wallpaper3.jpg
  ├── nom-personnage-2/
  │   ├── wallpaper1.jpg
  │   └── wallpaper2.png
  └── ...
```

## Instructions

1. Créez un dossier pour chaque personnage (ex: `naruto`, `goku`, `luffy`)
2. Placez tous les wallpapers de ce personnage dans son dossier
3. Formats supportés : `.jpg`, `.jpeg`, `.png`, `.gif`, `.webp`
4. Les noms des dossiers apparaîtront comme filtres dans la Gallery

## Exemple

```
public/wallpapers/
  ├── naruto/
  │   ├── naruto-1.jpg
  │   ├── naruto-2.png
  │   └── naruto-3.jpg
  ├── goku/
  │   ├── goku-1.jpg
  │   └── goku-2.png
  └── luffy/
      ├── luffy-1.jpg
      └── luffy-2.png
```

Les wallpapers seront automatiquement détectés et affichés dans la Gallery !
